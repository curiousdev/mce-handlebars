Handlebars Templating Plugin for TinyMCE.

This a opensource plugin for templating content within TinyMCE.

Demo

Setup

Depending on your framework, there are multiple ways to configure this plugin.

Start with these basic examples first. In general, you need to configure the tinymce.init() with 

  
    
tinymce.init({

  // Step 1: Configure the plugin to load and add it to the toolbar
  plugins: "handlebars code", // add handlebars plugin
  toolbar: "handlebars code", // add it to the toolbar

  // Step 2: Include the configuration for plugin prefix/suffix and items
  handlebars: {
    prefix: "{{",
    items: [
      { text: "First Name", value: "firstName" },
      { text: "Last Name", value: "lastName" },
      { text: "Hotel Name", value: "hotelName" },
      { text: "Checkin Date", value: "begins" },
      { text: "Checkout Date", value: "ends" },
    ],
    suffix: "}}",
  },

  // Step 3: Add any custom styling
  content_css: ["custom_affix.css"], // .handlebars-data-affix { font-weight: 900; color: rgb(46, 149, 211); }
});
